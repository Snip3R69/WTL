{{{{{In the HTML code above, we have a link with the class popup-link that will show a popup when it is hovered over. The popup has the ID my-popup and contains two text boxes and a button.

JavaScript (jQuery):}}}}

$(document).ready(function() {
	$(".popup-link").mouseover(function() {
		$("#my-popup").show();
	});
	$(".popup-link").mouseout(function() {
		$("#my-popup").hide();
	});
});
